# Your Story Press — Final Front-End Project

This is the final Vercel-ready version of the Your Story Press front-end project.

## Start file

Open `index.html`.

## Folder structure

```text
your-story-press-vercel-final/
├── index.html
├── assets/
```

## Features

- Landing page for Your Story Press
- Create Your Book interaction
- Story preview generation
- Save draft demo
- Submit book demo
- Login and registration demo
- Dashboard using browser localStorage
- Contact form demo
- Team CV pages
- Responsive CSS

## Important note

Login, registration, saved drafts, and submitted books are simulated using browser localStorage. This is acceptable for a front-end prototype, but a real website would need backend authentication and a database.
